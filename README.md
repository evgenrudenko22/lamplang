# LampLang
![GitHub License](https://img.shields.io/github/license/evgenrudenko22/lamplang)
## About
LampLang is simple statically typed, compiled programming language wroted on Rust.
## How it works?
LampLang compiler translate code to C. Next it build C code and link with standard library.
## Getting Started
You can write "Hello world" programe very easy on LampLang: ```
use io
print("{s}", "Hello, world")
```
this code print "Hello, world" to console
